App name: CGPA Calculator

Description: An academic calculator for keeping track of academic progress of the students.

Type of app: Web Application

Main features: 

-user can form an account, sign in or sign out
-add course related information
-view CGPA
-update course information
-delete those information
-view the CGPA of a particular semester
-sort courses by faculty, coursecode, etc.
-view the trends in which CGPAs change
-view transcript
-export transcript
-view/update the grading system

Operating system: Windows 

How to use: Extract the zip file and run the .exe file. Install XAMPP for html. Either Netbeans or Eclipse IDE. 

Written in: JavaScript, CSS, html, JQuery     

Tools/framework used: XAMPP, JVisual studio code 

References: I used code from a GitHub repository.
